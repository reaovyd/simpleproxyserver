\doxysubsection{Detailed Description}
This is the object that represents the connection pool. 

This is the object responsible for holding each of the user\textquotesingle{}s connection that can later be passed to an event handler like {\ttfamily select(2)} and, then, further processes the connections that correspond to some socket file descriptor.
