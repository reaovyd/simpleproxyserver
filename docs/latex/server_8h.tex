{\footnotesize\ttfamily int run\+\_\+proxy\+\_\+server (\begin{DoxyParamCaption}\item[{char $\ast$}]{port }\end{DoxyParamCaption})}



This starts up a server on the calling thread. 

It takes as argument a port which must be an integer value. The server will create a new TCP socket that binds to the port specified by the argument {\itshape port}. Then, the server will be set up to listen for any new TCP connections and accept such connections. A client can send any HTTP request and the server will handle the request by directing the request to the actual server the user wants to access. The function fails when any of the socket system calls, memory allocation calls, or any other system calls fail. It also fails if the port is not a short integer when parsed by the function.
